1️⃣ What is the difference between var, let, and const?
ans:var declared variables which is function-scoped and updates with the same scope and allow re-declaration.let declares variables which is block scope allow updates but not re-declaration .const declares the values which is constants cannot be changed.

2️⃣ What is the spread operator (...)?
ans:Spread operator is used to expand elements of an array or object into individual items.

3️⃣ What is the difference between map(), filter(), and forEach()?
ans:map() creates a new array applying a function to each element. filter() returns a new array containing only elements meeting condition.forEach() only loops through elements and does not return new array.

4️⃣ What is an arrow function?
ans:Arrow functions allow a shorter syntax for function expressions. example->
const add = (a,b)=>{
    return a + b;
}

5️⃣ What are template literals?
ans:Template literals use backticks `` to create strings.It allow both single and double quotes inside string also allow multiline strings.It allows embedding variables inside strings using ${}.
